# 🌍 TourismPulse – Travel Analytics Engine

## 📌 Project Overview

TourismPulse is a Travel Analytics Engine developed using Python, Pandas, and Streamlit. It analyzes tourism data to provide insights into seasonal travel trends, visitor preferences, destination popularity, and emerging tourism hotspots through an interactive dashboard.

---

## ✨ Features

- Tourism Overview Dashboard
- Seasonal Tourism Analysis
- Peak & Off-Peak Travel Analysis
- Visitor Preference Analysis
- Destination Popularity Ranking
- Tourism Demand Forecast
- Destination Comparison
- Emerging Hotspots Analysis

## ▶️ How to Run

1. Clone the repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
